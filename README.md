
# Tuxogotchi

An Virtual pet for your linux terminal!

## Installation

Install and start tuxogotchi with git:

```bash
  git clone https://github.com/grzesiowski1223/TerminalPetTux
  cd TerminalPetTux
  bash tuxogotchi.sh
```
